static void ADC1Init ( )
static

ADC1 Initialization Function.

Returns
None

Initializes ADC1 in continuous sampling mode. Only a single input is converted (temp sensor input channel). The sample time is set to 601 ADC cycles. The ADC is clocked from 72MHz core clock divided by 8. 601 cycles / (72MHz/8) -> 66us. The Temp sensor requires a minimum 10us setting time for accurate measurements.
