static void DMA_Init ( )
static

Init all DMA channels in use and configure DMA interrupts.

Returns
void

Currently use three DMA channels, on DMA peripheral 1: DMA2/3 - IMU SPI port, used for burst reads from IMU DMA5 - User SPI port, Used for buffer burst outputs. Rx not used

All DMA channels are configured to generate interrupts upon the completion of a transfer. The user SPI DMA interrupt is used just to check for any error events which might have occurred during the burst read process.

The IMU DMA interrupts are used to determine when a burst read from the IMU has completed. Both the Tx and Rx channels must be done for a burst read to have completed (Tx should finish last).
