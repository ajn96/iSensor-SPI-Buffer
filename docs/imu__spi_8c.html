◆ InitImuCsTimer()

void InitImuCsTimer ( )

Inits TIM3 for use in PWM mode to drive CS.

Returns
void

TIM4 is used to drive SPI buffered data acquisition from the IMU. One timer interrupt is generated per SPI word clocked out from the DUT. TIM4 runs at a full 72MHz. With a 16-bit resolution and a time base of 72MHz, TIM4 will roll over every 910us The worst case spi period allowed is 255us stall + (17 bits / 140KHz) -> 376us
