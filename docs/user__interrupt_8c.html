◆ UpdateDIOConfig()

void UpdateDIOConfig ( )

Validates DIO_CONFIG settings and applies to GPIO.

General procedure is as follows:

passPins Set SW_IN1 - SW_IN4 output values based on passPins. These act as inputs to the ADG1611 analog switch. For each bit set in passPins, configure the corresponding DIOx_Slave signal as an input (tristate).

intPins Configure selected interrupt pins as output GPIO

overflowPins Configure selected overflow interrupt pins as output GPIO

Any pins which are unused will be configured as inputs.
